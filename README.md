# ext-ct-snotes - archived!

An attempt to bring the functionality of the [Snotes application](https://github.com/scorbo2/snotes/)
into the [CryptText application](https://github.com/scorbo2/crypttext/) as an application extension.

---

This repo is archived and is officially abandoned.

Lessons learned:

- CryptText is a simple text editor with built-in encryption options
- Snotes is a full-blown note management application with many rules regarding how and where notes should be saved.
- The highly opinionated nature of Snotes makes it very difficult (impossible even?) to implement as an extension to the
  CryptText application.
- Weirdly, it would probably be much easier to implement CryptText's encryption features as an extension to the Snotes
  application, rather than the other way around.
- This project was well-intentioned, but mostly failed. Managed to salvage some code out of here to use in Snotes 2.0
  though, so it wasn't a total loss.

There will be no further updates! This repo is abandoned.
